# SQL Data Analysis Task (Internship)

This repository contains SQL queries and analysis for the assigned data analysis task.

## 📁 Folder Structure
- **dataset/** – contains ecommerce.sql dataset  
- **queries/** – contains SQL queries  
- **screenshots/** – store outputs  
- **ecommerce.db** – auto-created SQLite database  

## ▶ Database Setup
\\\
sqlite3 ecommerce.db < dataset/ecommerce.sql
\\\

## ▶ Run Queries
Open **task3_queries.sql** and execute in SQLite or VS Code SQLTools.
